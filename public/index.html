API de Videojuegos

Una API RESTful para gestionar videojuegos y desarrolladoras

Información de la API

Esta API permite gestionar videojuegos y desarrolladoras. Puedes crear, leer, actualizar y eliminar videojuegos y desarrolladoras.

Advertencia

Para añadir nuevas desarrolladoras y videojuegos, es necesario autenticarse con un token JWT.

Endpoints Disponibles

Usuarios

  • POST /api/usuarios/login - Iniciar sesión 
    POST /api/usuarios/login

{
  "email": "usuario@ejemplo.com",
  "password": "contraseña123"
}
  • POST /api/usuarios/registro - Registrar un nuevo usuario 
    POST /api/usuarios/registro

{
  "nombre": "Usuario Ejemplo",
  "email": "usuario@ejemplo.com",
  "password": "contraseña123"
}
  • GET /api/usuarios/perfil - Obtener perfil de usuario (requiere autenticación) 
    GET /api/usuarios/perfil
Autorización: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
  • GET /api/usuarios/ - Obtener todos los usuarios 
    GET /api/usuarios/
